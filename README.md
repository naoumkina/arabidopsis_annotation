# NCBI Annotation Pipeline

1. Create and activate the conda environment

```bash
conda create -n arabidopsis-annot python=3.11 -y
conda activate arabidopsis-annot

# Install required packages
conda install -c conda-forge biopython pandas lxml requests -y

# Optional but recommended for documentation
conda install -c conda-forge jupyterlab -y
```
Why this matters: 
A dedicated environment ensures reproducibility and prevents conflicts with system Python or cluster modules.

2. Recommended project directory structure

```
~/projects/arabidopsis_annotation/
│
├── agi_list.txt
├── annotate_arabidopsis_from_ncbi.py
├── results/      # output files generated by the script
└── notebooks/    # JupyterLab documentation (optional)
```

3. Run the annotation script
From inside the project directory:

```bash
python annotate_arabidopsis_from_ncbi.py
```

4. (Optional) Preview results inside JupyterLab

```Python
import pandas as pd

df = pd.read_csv("results/ncbi_annotation_for_arabidopsis_genes.csv")
df.head()
```

5. (Optional) Display the script inside the notebook

```Python
%load annotate_arabidopsis_from_ncbi.py
```
